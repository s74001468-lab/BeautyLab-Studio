# Настройка Google Sheets API и Service Account

Чтобы бот мог писать данные о записях в вашу Google Таблицу `BeautyLab`, нужно создать сервисный аккаунт в Google Cloud. Следуй этой пошаговой инструкции:

## Шаг 1: Создание проекта в Google Cloud
1. Перейди на консоль [Google Cloud Console](https://console.cloud.google.com/).
2. Авторизуйся под своим Google-аккаунтом.
3. Нажми **Select a project** в верхней панели (или на название текущего проекта) и затем **New Project**.
4. Назови проект (например, `beautylab-bot`) и нажми **Create**. Обрати внимание: на создание уйдет несколько секунд. Выбери этот проект после создания.

## Шаг 2: Включение Google Sheets API и Google Drive API
1. В левом меню выбери **APIs & Services** -> **Library**.
2. В строке поиска введи "Google Sheets API", выбери его и нажми **Enable** (Включить).
3. Вернись в **Library**, введи "Google Drive API", выбери его и также нажми **Enable**. (Drive нужен для гибких прав доступа таблицы).

## Шаг 3: Создание сервисного аккаунта и ключа
1. В левом меню перейди в **APIs & Services** -> **Credentials**.
2. Нажми наверху **+ CREATE CREDENTIALS** и выбери **Service account**.
3. Введи имя (например, `beautylab-db`) и нажми **Create and Continue**, затем **Done** (роли можно не назначать).
4. В появившемся списке сервисных аккаунтов нажми на email свежесозданного аккаунта (он выглядит как `название@имя-проекта.iam.gserviceaccount.com`). *Обязательно скопируй этот email — он нам еще понадобится!*
5. Перейди на вкладку **KEYS**.
6. Нажми **Add Key** -> **Create new key**. 
7. Выбери формат **JSON** и нажми **Create**.
8. Файл скачается на твой компьютер. Переименуй его в **`credentials.json`** и перемести в папку нашего проекта: `/beautylab-bot/credentials.json`.
   > **ВАЖНО:** Этот файл содержит секретный ключ, не делись им ни с кем. Этот файл будет добавлен в `.gitignore`, чтобы он не улетел в GitHub!

## Шаг 4: Настройка самой Google Таблицы
1. Зайди в обычный Google Drive под своим аккаунтом и создай новую Google Таблицу (или открой существующую).
2. Назови таблицу, например, `BeautyLab Bookings`.
3. В первой строке (заголовки) напиши следующие столбцы:
   `ID`, `Telegram ID`, `Имя клиента`, `Телефон`, `Услуга`, `Мастер`, `Дата`, `Время`, `Длительность`, `Комментарий`, `Статус`, `Created_at`.
4. Нажми кнопку **Настройки доступа** (Share) в правом верхнем углу.
5. В поле "Добавьте пользователей" вставь **email сервисного аккаунта**, который мы скопировали на Шаге 3 (п.4).
6. Выдай права **Редактор** (Editor) и нажми "Поделиться" (отправлять уведомление не нужно, сними галочку).

## Шаг 5: Получение SPREADSHEET_ID для `.env`
1. Посмотри на URL-адрес открытой таблицы, он выглядит так:
   `https://docs.google.com/spreadsheets/d/1BxiMVs0X8922245354y0112344xJk/edit#gid=0`
2. Скопируй набор символов между `/d/` и `/edit`.
   В примере выше это: `1BxiMVs0X8922245354y0112344xJk`.
3. Вставь это значение в файл `.env` напротив переменной `SPREADSHEET_ID`.

🎉 **Готово!** Теперь у бота есть ключи от квартиры, где лежат записи клиентов.
